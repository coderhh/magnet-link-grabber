Privacy Policy for Magnet Link Grabber

Last Updated:

TL;DR: This extension does not collect, store, or transmit any user data. All processing happens locally in your browser.

1. Data Collection

We do not collect any user data.

Magnet Link Grabber does not collect, store, or transmit any personal information, browsing history, or user data. The extension operates entirely locally within your browser.

2. How the Extension Works

The extension:

  • Scans the current webpage for magnet links when you click the extension icon
  • Processes the page content locally in your browser
  • Displays found magnet links in the popup interface
  • Copies selected links to your clipboard when you choose to do so

All of this processing happens entirely on your device. No data is sent to any external servers.

3. Permissions Used

The extension uses the following permissions:

  • activeTab: To access the current tab's content when you click the extension icon
  • scripting: To inject a content script that scans for magnet links
  • Host permissions (<all_urls>): To allow scanning any webpage you visit for magnet links

These permissions are only used for the extension's core functionality and are not used to collect or transmit data.

4. Third-Party Services

This extension does not use any third-party services, analytics, or tracking tools. No external servers are contacted.

5. Data Storage

The extension does not store any data. No information is saved to your device or any external location.

6. Changes to This Policy

If we make any changes to this privacy policy, we will update the "Last Updated" date at the top of this page. Since we do not collect data, this policy is unlikely to change.

7. Contact

If you have any questions about this privacy policy, please contact us through the Chrome Web Store listing.

This extension is committed to user privacy. We believe in transparency and user control.
